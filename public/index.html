What are the inputs and where can I get these input keys?

  1. Device Id -> This key is cruical for sending message to device and you can get that key in the Azure IoT Hub center
  2. Service Key, Consumer Group -> These keys are crucial for reading messages that comes to your Azure IoT Hub Center, you can get that keys from azure IoT Hub Center too.

Possible Errors

  1. If you can't send a message to the Azure IoT Hub, you wrote the wrong keys at the beginning possibly, please check your keys and don't add quotes to the keys.
  2. Remember, this application sends messages to the Azure IoT Hub, Therefore, we need to connect our device to the hub via the desktop application that we will create in the next tutorial part, and listen if the message income. If your device is not connected to the Hub, It can't get that messages.
